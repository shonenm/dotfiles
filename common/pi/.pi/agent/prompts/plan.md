# Plan

Create a step-by-step implementation plan for the requested change.

## Output
1. **Goal**: One-sentence summary of what we're building
2. **Approach**: High-level strategy and key decisions
3. **Steps**: Numbered, small, independently verifiable tasks
4. **Risks**: What could go wrong and how to mitigate
5. **Verification**: How to confirm each step works

Keep the plan in a file (TODO.md or docs/agent-plan.md) and reference it during implementation.
